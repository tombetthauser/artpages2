Using an Existing Website Name:

This section is for if you already have a website domain name registered, it's super easy to continue using it but you have to do some light computer hacking. If any of this gives you a headache I can do all of this for you super quickly so don't hesitate to toss it to me if you're suffering at all.



If You Feel Comfortable Doing This:

First you need to login to whatever website you pay for the domain name, this might be Dream Host, Godaddy, Google Domains etc. Once your logged in you need to go into the section where you can manage your domain names and then click on any section lebelled with DNS.

Once you're in the DNS Management section for your domain name find the Records section and add two neew A Records. Also you'll want to delete any existing A Records.

A Records you'll need to add:

  1. @ 192.30.252.153
  2. @ 192.30.252.154
